# LSTM模型与交易策略图表解释报告

## 1. 报告用途

本报告解释 LSTM 股价方向预测、滚动验证和交易策略相关图表的用途、读图方法和当前结论。它用于配合 `outputs/lstm_report.md` 和 `outputs/lstm_rolling_report.md`，说明模型为什么这样选、策略为什么这样交易，以及普通模型和融合模型的差异。

## 2. 单次样本外测试图表

### 2.1 `outputs/figures/lstm_prediction.png`

**用途**：展示测试集真实方向、模型预测方向和上涨概率的时间变化。

**怎么读**：如果预测概率在上涨前明显抬升，并在下跌段回落，说明模型捕捉到了一定趋势信号；如果概率长期贴近阈值或单边偏高/偏低，说明模型更像是在做保守分类。

**当前结论**：普通 LSTM 的概率信号更适合作为交易阈值输入；融合模型在短周期上容易变得更保守。

### 2.2 `outputs/figures/lstm_price_direction.png`

**用途**：把预测信号叠加到收盘价走势上，观察买入信号是否出现在较合理的位置。

**怎么读**：买入信号应尽量集中在上升段或震荡后的突破段；如果信号密集出现在持续下跌段，说明策略风险较高。

**当前结论**：加入双阈值和行业过滤后，信号数量明显减少，更适合解释为“高置信度持仓”，而不是每天都给出交易动作。

### 2.3 `outputs/figures/lstm_strategy_return.png`

**用途**：比较 LSTM 策略、买入持有和朴素动量策略的累计收益。

**怎么读**：这张图是判断模型是否有交易价值的主图。若模型曲线高于买入持有且回撤更浅，说明预测信号经过策略层后有实际意义。

**当前结论**：普通 LSTM 在本轮测试中优于买入持有和朴素动量；融合 LSTM 没有体现出稳定交易优势，因此不建议作为当前主策略。

### 2.4 `outputs/figures/lstm_metrics_comparison.png`

**用途**：横向比较 Accuracy、F1-score、策略收益、回撤等指标。

**怎么读**：不能只看 Accuracy。若模型 Accuracy 较高但 F1-score 为0或交易次数为0，说明它可能主要通过预测“非上涨”获得准确率，交易价值有限。

**当前结论**：普通 LSTM 的综合表现更均衡；融合模型需要在更长预测周期或更多股票样本上继续验证。

### 2.5 `outputs/figures/lstm_confusion_matrix.png`

**用途**：展示上涨/非上涨分类的混淆矩阵。

**怎么读**：重点看上涨样本是否被识别出来。若真实上涨被大量预测为非上涨，模型会偏向空仓；若非上涨被大量预测为上涨，策略回撤会变大。

**当前结论**：混淆矩阵用于解释为什么 F1-score 比 Accuracy 更重要，尤其适合说明模型是否存在“少交易但看似准确”的问题。

## 3. 滚动验证图表

### 3.1 `outputs/figures/lstm_rolling_f1.png`

**用途**：展示不同年度测试折中的 F1-score。

**怎么读**：如果某模型只在单一年份表现好，而其他年份明显失效，说明稳定性不足；如果多数年份保持相对优势，才适合作为主模型。

**当前结论**：普通 LSTM 在滚动验证中更适合作为主模型；融合模型的方向识别能力不够稳定。

### 3.2 `outputs/figures/lstm_rolling_strategy_return.png`

**用途**：展示每个测试年份扣除交易成本后的策略收益。

**怎么读**：这张图比 F1-score 更接近真实策略判断。模型必须在多个年份中尽量避免深亏，而不只是某一年赚得多。

**当前结论**：普通 LSTM 的平均策略收益更好，但仍存在年度差异；后续应继续做跨股票和更严格交易成本检验。

## 4. 最新滚动信号图表

### 4.1 `outputs/figures/rolling_prediction_signals_base.png`

**用途**：展示普通 LSTM 最新滚动模型给出的买入/空仓信号。

**怎么读**：图中的买入点代表模型概率超过买入阈值且行业过滤通过；没有买入点并不表示看空，而是表示信号置信度不足。

**当前结论**：这张图应作为当前最终策略信号的主解释图。

### 4.2 `outputs/figures/rolling_prediction_signals_fusion.png`

**用途**：展示融合 LSTM 最新滚动模型给出的信号。

**怎么读**：它主要用于和普通 LSTM 对照，观察经营绩效和TOPSIS变量是否改变短期交易判断。

**当前结论**：融合模型更适合作为研究对照，不建议独立作为当前5日交易主模型。

## 5. 汇报时的推荐讲法

第一步先用 `lstm_strategy_return.png` 说明：模型评价不能只看方向准确率，最终要看扣除成本后的策略结果。

第二步用 `lstm_metrics_comparison.png` 和 `lstm_confusion_matrix.png` 解释：融合模型虽然加入更多信息，但在短周期上可能变得过于保守，普通 LSTM 更适合当前任务。

第三步用 `lstm_rolling_strategy_return.png` 说明：滚动验证比单次切分更接近真实样本外检验，因此模型选择应以滚动结果为主。

第四步用 `rolling_prediction_signals_base.png` 给出最终策略信号：当前信号为持仓或空仓，并解释阈值、行业过滤和交易成本假设。

## 6. 使用注意

这些图表用于建模研究和策略回测说明，不构成投资建议。由于样本来自单只股票，且交易成本、滑点、涨跌停、停牌和资金容量约束仍需进一步细化，最终结论应以多股票滚动验证和更严格回测为准。
